循环神经网络

2020-08-15 16 min read

一、算法背景

循环神经网络（Recurrent Neural Network, RNN） 是一类以序列（sequence）数据为输入，在序列的演进方向进行递归（recursion）且所有节点（循环单元）按链式连接的递归神经网络（recursive neural network）。

对循环神经网络的研究始于二十世纪80-90年代，并在二十一世纪初发展为深度学习（deep learning）算法之一 ，其中双向循环神经网络（Bidirectional RNN, Bi-RNN）和长短期记忆网络（Long Short-Term Memory networks，LSTM）是常见的的循环神经网络。

RNN（Recurrent Neural Networks）循环神经网络是一种节点定向连接成环的人工神经网络。这种网络的内部状态可以展示动态时序行为。不同于前馈神经网络的是，RNN可以利用它内部的记忆来处理任意时序的输入序列，这让它可以更容易地处理如不分段的手写识别和语音识别信息等。RNN不仅会学习当前时刻的信息，也会依赖之前的序列信息。由于其特殊的网络模型结构解决了信息保存的问题，所以 RNN 对处理时间序列和语言文本序列问题有独特的优势。

二、算法原理

 和传统的神经网络一样，循环神经网络的基本结构也分为三个部分：输入层、隐藏层和输出层，不同的是循环神经网络的隐藏层中多了一个环（循环层），并且是以序列数据作为输入。一个简单的结构如下图所示：

 图中U代表输入层到隐藏层的权重，W代表隐藏层之间的权重，V代表隐藏层到输出层的权重。循环神经网络的这一个环正是它为什么能解决序列问题的原因，那么这个环在网络结构中到底是怎么发挥作用的，其实上面给出的网络结构图是很抽象的，很难能看出其中的原理，所以我们要把这个结构按时间线进行展开：

 上图中，xt−1x_{t-1}xt−1​、xtx_{t}xt​、xt+1x_{t+1}xt+1​分别是t-1、t、t+1时刻的输入，st−1s_{t-1}st−1​、sts_{t}st​、st+1s_{t+1}st+1​分别是t-1、t、t+1时刻的隐藏层值，ot−1o_{t-1}ot−1​、oto_{t}ot​、ot+1o_{t+1}ot+1​分别是t-1、t、t+1时刻的输出值。根据展开图我们可以看到，在t时刻sts_{t}st​的值不仅与xtx_{t}xt​有关系，还和st−1s_{t-1}st−1​有关系，而隐藏层中的值又被称为RNN的记忆，通过上面的结构我们可以看到，每个时刻的s值都是互相关联的，也就是说RNN的记忆之间相互构成关系，这也就是RNN具有记忆功能的原因。

三、循环神经网络的后向传播（BPTT）

前面我们已经介绍了RNN的前向传播原理，下面我们开始学习RNN的反向传播。
 不考虑偏差项，我们知道：

ht=ϕ(U⋅xt+W⋅ht−1)h_{t}=\phi(U·x_{t}+W·h_{t-1}) ht​=ϕ(U⋅xt​+W⋅ht−1​)

ot=softmax(V⋅ht)o_{t}=soft max (V·h_{t}) ot​=softmax(V⋅ht​)

 t时刻的输入xtx_{t}xt​对应的真实值为yty_{t}yt​，给定每个时刻损失函数计算公式为ℓ\ellℓ,其值用小写lll表示，长度为T的整个时序数据的损失函数用L表示，得：

l=ℓ(ot,yt)l=\ell(o_{t},y_{t}) l=ℓ(ot​,yt​)

L=1T∑t=1Tℓ(ot,yt)L=\frac{1}{T}\sum^{T}_{t=1}\ell(o_{t},y_{t}) L=T1​t=1∑T​ℓ(ot​,yt​)

 我们需要学习的参数有V、W和U，使用梯度下降的方法，假设学习率为η\etaη：

∇V=∂L∂V\nabla V=\frac{∂L}{∂V} ∇V=∂V∂L​

∇W=∂L∂W\nabla W=\frac{∂L}{∂W} ∇W=∂W∂L​

∇U=∂L∂U\nabla U=\frac{∂L}{∂U} ∇U=∂U∂L​

V=V−η∂L∂VV=V-\eta\frac{∂L}{∂V} V=V−η∂V∂L​

W=W−η∂L∂WW=W-\eta\frac{∂L}{∂W} W=W−η∂W∂L​

U=U−η∂L∂UU=U-\eta\frac{∂L}{∂U} U=U−η∂U∂L​

 首先，目标函数有关各时刻输出层变量的梯度可以很容易计算出来：

∂L∂ot=∂ℓ(ot,yt)T⋅∂ot\frac{∂L}{∂o_{t}}=\frac{∂\ell(o_{t},y_{t})}{T·∂o_{t}} ∂ot​∂L​=T⋅∂ot​∂ℓ(ot​,yt​)​

 这个时候我们发现，现在已经可以计算∇V\nabla V∇V。因为V是经过o1o_{1}o1​，…，oTo_{T}oT​通向L，根据链式法则：

∇V=∂L∂V=∑t=1T∂L∂ot⋅∂ot∂V=∑t=1T∂L∂otht=∑t=1T∂ℓ(ot,yt)T⋅∂otht\nabla V=\frac{∂L}{∂V}=\sum^{T}_{t=1}\frac{∂L}{∂o_{t}}·\frac{∂o_{t}}{∂V}=\sum^{T}_{t=1}\frac{∂L}{∂o_{t}}h_{t}=\sum^{T}_{t=1}\frac{∂\ell(o_{t},y_{t})}{T·∂o_{t}}h_{t} ∇V=∂V∂L​=t=1∑T​∂ot​∂L​⋅∂V∂ot​​=t=1∑T​∂ot​∂L​ht​=t=1∑T​T⋅∂ot​∂ℓ(ot​,yt​)​ht​

 有了∇V\nabla V∇V之后，我们还要计算∇U\nabla U∇U和∇W\nabla W∇W。根据RNN的结构图我们知道，U和W通向L的过程中都要经过隐藏层，所以我么需要先计算∂L∖∂ht∂L\setminus∂h_{t}∂L∖∂ht​：
 由于∂L∖∂ht∂L\setminus∂h_{t}∂L∖∂ht​计算比较复杂，所以我么先看最终时刻T，hTh_{T}hT​只通过oTo_{T}oT​通向L，我们可以得到：

∂L∂hT=∂L∂oT⋅∂oT∂hT=∂L∂oT⋅V\frac{∂L}{∂h_{T}}=\frac{∂L}{∂o_{T}}·\frac{∂o_{T}}{∂h_{T}}=\frac{∂L}{∂o_{T}}·V ∂hT​∂L​=∂oT​∂L​⋅∂hT​∂oT​​=∂oT​∂L​⋅V

 接下来，对于时刻t<Tt<Tt<T，由于hth_{t}ht​经过ht+1h_{t+1}ht+1​和oto_{t}ot​通向L，根据链式法则，我们可以得到：

∂L∂ht=∂L∂ht+1⋅∂ht+1∂ht+∂L∂ot⋅∂ot∂ht=∂L∂ht+1⋅W+∂L∂ot⋅V\frac{∂L}{∂h_{t}}=\frac{∂L}{∂h_{t+1}}·\frac{∂h_{t+1}}{∂h_{t}}+\frac{∂L}{∂o_{t}}·\frac{∂o_{t}}{∂h_{t}}=\frac{∂L}{∂h_{t+1}}·W+\frac{∂L}{∂o_{t}}·V ∂ht​∂L​=∂ht+1​∂L​⋅∂ht​∂ht+1​​+∂ot​∂L​⋅∂ht​∂ot​​=∂ht+1​∂L​⋅W+∂ot​∂L​⋅V

 由递归关系我们可以得到：

∂L∂ht=∂L∂ht+2⋅W2+∂L∂ot+1⋅W⋅V+∂L∂ot⋅V\frac{∂L}{∂h_{t}}=\frac{∂L}{∂h_{t+2}}·W^{2}+\frac{∂L}{∂o_{t+1}}·W·V+\frac{∂L}{∂o_{t}}·V ∂ht​∂L​=∂ht+2​∂L​⋅W2+∂ot+1​∂L​⋅W⋅V+∂ot​∂L​⋅V

∂L∂ht=∂L∂ht+3⋅W3+∂L∂ot+2⋅W2⋅V+∂L∂ot+1⋅W⋅V+∂L∂ot⋅V\frac{∂L}{∂h_{t}}=\frac{∂L}{∂h_{t+3}}·W^{3}+\frac{∂L}{∂o_{t+2}}·W^{2}·V+\frac{∂L}{∂o_{t+1}}·W·V+\frac{∂L}{∂o_{t}}·V ∂ht​∂L​=∂ht+3​∂L​⋅W3+∂ot+2​∂L​⋅W2⋅V+∂ot+1​∂L​⋅W⋅V+∂ot​∂L​⋅V

………… ……

 对于任意1≤t≤T1\le t\le T1≤t≤T,我们可以得到目标函数关于隐藏层变量梯度的通项公式：

∂L∂ht=∑i=tT∂L∂oT+t−iWT−iV\frac{∂L}{∂h_{t}}=\sum^{T}_{i=t}\frac{∂L}{∂o_{T+t-i}}W^{T-i}V ∂ht​∂L​=i=t∑T​∂oT+t−i​∂L​WT−iV

 到此，我么已经成功的求出了目标函数关于隐藏层变量的梯度，可是我们发现，当每个时序训练数据样本的时序长度T较大或者时刻较小时，该梯度容易出现衰减或爆炸的问题。

 有了隐藏层变量的梯度之后，我们就可以来求∇U\nabla U∇U和∇W\nabla W∇W，上面说过U和W通向L的过程中都要经过隐藏层h1h_{1}h1​，…，hTh_{T}hT​，根据链式法则：

∇U=∂L∂U=∑t=1T∂L∂ht⋅∂ht∂U=∑t=1T∂L∂htxt=∑t=1T(xt⋅∑i=tT∂L∂oT+t−iWT−iV)\nabla U=\frac{∂L}{∂U}=\sum^{T}_{t=1}\frac{∂L}{∂h_{t}}·\frac{∂h_{t}}{∂U}=\sum^{T}_{t=1}\frac{∂L}{∂h_{t}}x_{t}=\sum^{T}_{t=1}(x_{t}·\sum^{T}_{i=t}\frac{∂L}{∂o_{T+t-i}}W^{T-i}V) ∇U=∂U∂L​=t=1∑T​∂ht​∂L​⋅∂U∂ht​​=t=1∑T​∂ht​∂L​xt​=t=1∑T​(xt​⋅i=t∑T​∂oT+t−i​∂L​WT−iV)

∇W=∂L∂U=∑t=1T∂L∂ht⋅∂ht∂W=∑t=1T∂L∂htht−1=∑t=1T(ht−1⋅∑i=tT∂L∂oT+t−iWT−iV)\nabla W=\frac{∂L}{∂U}=\sum^{T}_{t=1}\frac{∂L}{∂h_{t}}·\frac{∂h_{t}}{∂W}=\sum^{T}_{t=1}\frac{∂L}{∂h_{t}}h_{t-1}=\sum^{T}_{t=1}(h_{t-1}·\sum^{T}_{i=t}\frac{∂L}{∂o_{T+t-i}}W^{T-i}V) ∇W=∂U∂L​=t=1∑T​∂ht​∂L​⋅∂W∂ht​​=t=1∑T​∂ht​∂L​ht−1​=t=1∑T​(ht−1​⋅i=t∑T​∂oT+t−i​∂L​WT−iV)

import numpy as np
from functools import reduce
# reduce：在python3中不是内置函数，需要import functools调用
# reduce()函数会对参数序列中元素进行累积。reduce(function, iterable[, initializer])   (函数，可迭代对象(,初始参数))
# 例如： reduce(lambda x, y: x+y, [1,2,3,4,5])      结果为：15

定义激活函数

def element_wise_op(array, op):         #中间函数
    #np.nditer详细介绍：https://www.runoob.com/numpy/numpy-terating-over-array.html
    for i in np.nditer(array,op_flags=['readwrite']):  #对矩阵array进行修改
        i[...] = op(i)

class ReluActivator(object):            #Relu激活函数
    def forward(self, weighted_input):
        return max(0, weighted_input)   #return weighted_input
    def backward(self, output):
        return 1 if output > 0 else 0

class IdentityActivator(object):        #恒等激活函数
    def forward(self, weighted_input):
        return weighted_input
    def backward(self, output):
        return 1

定义循环层

class RecurrentLayer(object):           #循环层
    #调用该类时自动执行 _init_ 函数
    def __init__(self, input_width, state_width,activator, learning_rate): #输入数据的维度、隐藏层维度、激活函数、学习率
        self.input_width = input_width
        self.state_width = state_width
        self.activator = activator
        self.learning_rate = learning_rate
        self.times = 0      #初始时刻t=0
        self.state_list = [] 
        self.state_list.append(np.zeros((state_width, 1)))  #初始化隐藏层为0，2行1列      
        self.U = np.random.uniform(-1e-4, 1e-4,(state_width, input_width))  #初始化权重矩阵U，2行3列
        self.W = np.random.uniform(-1e-4, 1e-4,(state_width, state_width))  #初始化权重矩阵W，2行2列

    def forward(self, input_array):  #前向传播
        self.times += 1
        state = (np.dot(self.U, input_array) + np.dot(self.W, self.state_list[-1])) #计算当前时刻隐层的值,维度为2行1列
        element_wise_op(state, self.activator.forward)  #使用Relu激活函数激活当前时刻隐藏层的值
        self.state_list.append(state)  #保存当前时刻隐层的值

    def backward(self, sensitivity_array, activator):   #反向传播
        self.calc_delta(sensitivity_array, activator)   
        self.calc_gradient()
    
    #########计算误差#######
    def calc_delta(self, sensitivity_array, activator):  
        self.delta_list = []    #误差列表
        for i in range(self.times):
            self.delta_list.append(np.zeros((self.state_width, 1)))  #初始化为2行1列的全为0的矩阵
        self.delta_list.append(sensitivity_array)     
        for k in range(self.times - 1, 0, -1):   #从（self.times - 1）到0，不包括0，步长为 -1
            state = self.state_list[k+1].copy()    #复制隐层最后时刻的值
            element_wise_op(self.state_list[k+1],activator.backward)  #Relu激活函数反向传播
            self.delta_list[k] = np.dot(  np.dot(self.delta_list[k+1].T,self.W) , np.diag(state[:,0])  ).T   #计算误差    

    #########计算梯度########
    def calc_gradient(self):  
        self.gradient_list = []    #梯度列表
        for t in range(self.times + 1):
            self.gradient_list.append( np.zeros((self.state_width, self.state_width)) )  #初始化为0，2行2列
        for t in range(self.times, 0, -1):    #从 self.times 到0，不包括0，步长为 -1
            gradient = np.dot( self.delta_list[t] , self.state_list[t-1].T ) #计算t时刻的梯度
            self.gradient_list[t] = gradient
        self.gradient = reduce(lambda a, b: a + b, self.gradient_list, self.gradient_list[0])  #计算梯度总和

    def update(self):   #更新权重
        self.W -= self.learning_rate * self.gradient    
    
    def reset_state(self):    #重置隐层值
        self.times = 0      
        self.state_list = [] 
        self.state_list.append( np.zeros((self.state_width, 1)) )

初始化数据

def data_set():   #初始化输入矩阵和真实值矩阵
    x = [np.array([[1], 
                [2], 
                [3]]),
        np.array([[2], 
                [3], 
                [4]])]
    d = np.array([[1], 
                [2]])
    return x, d

定义test函数

def test():
    l = RecurrentLayer(3, 2, ReluActivator(), 1e-3)  #输入数据的维度、隐藏层维度、激活函数、学习率
    x, d = data_set()               #初始化输入矩阵和真实值矩阵
    l.forward(x[0])                 #t=0时刻传入数据x[0]      
    l.forward(x[1])                 #t=1时刻传入数据x[1] 
    l.backward(d, ReluActivator())
    return l

定义梯度检查函数

def gradient_check():       #梯度检查
    error_function = lambda o: o.sum()     
    rl = RecurrentLayer(3, 2, IdentityActivator(), 1e-3)
    x, d = data_set()
    rl.forward(x[0])
    rl.forward(x[1])

    sensitivity_array = np.ones(rl.state_list[-1].shape,dtype=np.float64)
    rl.backward( sensitivity_array , IdentityActivator() )

    epsilon = 10e-4  
    for i in range(rl.W.shape[0]):     #rl.W.shape[0]=2
        for j in range(rl.W.shape[1]):   #rl.W.shape[1]=2
            rl.W[i,j] += epsilon
            rl.reset_state()  #重置为0
            rl.forward(x[0])
            rl.forward(x[1])
            err1 = error_function(rl.state_list[-1])
            
            rl.W[i,j] -= 2*epsilon
            rl.reset_state()  #重置为0
            rl.forward(x[0])
            rl.forward(x[1])
            err2 = error_function(rl.state_list[-1])
            
            expect_grad = (err1 - err2) / (2 * epsilon)
            rl.W[i,j] += epsilon
            print ('weights(%d,%d): expected - actural %f - %f' % (i, j, expect_grad, rl.gradient[i,j]))

主函数

if __name__ == '__main__':
    test()
    gradient_check()
